Controls

Left, Right, A, D keys and the mouse moves the Dropper side to side.
Spacebar and the left mouse button drops the ball.
Enter key starts a new game after a gameover.
F key toggles the framerate display.
T key toggles the tuning window. This is a debugging tool that changes how the game behaves.

NOTE: You might have to zoom out to get the see the entire game. Zoom In: Control + Shift + =
Zoom Out: Control + Shift + -
